# AccessFox Demo — Manual Recording Checklist

## Pre-recording
- [ ] `npm run dev` running at http://localhost:3000
- [ ] App loads — AccessFox header visible
- [ ] Clicked button once to confirm results render
- [ ] Browser window set to clean size (hide bookmarks bar)
- [ ] `submission/narration.aiff` open in QuickTime, ready to play
- [ ] Screen recording tool ready (Cmd+Shift+5)

- [ ] Show **Why Now** field

## Post-recording

- [ ] Export as MP4
